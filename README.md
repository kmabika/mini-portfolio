## Getting Started

### Prerequisites
Make sure you have [Node](https://nodejs.org/en/) and [git](https://git-scm.com/) installed.

* pnpm
  ```sh
  npm install -g pnpm
  ```

## How to ...

### ... run locally?

- Clone the repo:

  ```bash
  git clone https://github.com/kmabika/mini-portfolio
  cd mini-portfolio
  ```

- Install the dependencies:

  ```bash
  pnpm install
  ```

- Fire up a development server:

  ```bash
  pnpm dev
